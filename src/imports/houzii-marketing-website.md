Figma Prompt — Houzii Marketing Website

Design a modern real estate marketplace marketing website for a platform called Houzii. The website introduces the Houzii ecosystem and helps users search for properties, invest in real estate, and access related services.

The design should feel modern, premium, trustworthy, and clean, similar to leading property platforms.

Use the following brand tokens:

Primary color: #7B2D42 (Maroon)
Accent color: #C94B60 (Rose)
Primary dark: #5E1F32
Font: Inter
Buttons: pill-shaped
Card radius: 16px

Style inspiration: modern real estate marketplace UI similar to Airbnb-style search layouts, with large imagery, soft shadows, clean cards, and clear typography.

Create the website with the following top navigation structure.

Main Navigation:

Home

Secondary Navigation:

The navigation should be minimal and modern, with the primary CTA Get Started highlighted.
