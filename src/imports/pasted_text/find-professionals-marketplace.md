esign the "Find Professionals" Marketplace Page for Houzii

Design a Find Professionals marketplace page for the Houzii platform where users can search and discover verified real estate-related professionals such as agents, lawyers, surveyors, interior designers, movers, inspectors, and property service providers.

The page should follow the same layout and interaction patterns used in the existing Explore Properties page, ensuring visual consistency with the Houzii website.

Use the same typography, spacing, cards, filters, and UI components already established on the platform.

Page Title

Find Professionals

Subtitle:

Connect with verified real estate professionals including agents, lawyers, surveyors, interior designers, movers, and other trusted service providers.

1. Search Section (Use Explore Page Pattern)

Create a search bar similar to the property explore page, but adapted for professionals.

Search fields should include:

• Service Type (dropdown or searchable input)
• Location (city, area, or state)
• Keyword search (name or specialization)

Example placeholder text:

Search professionals (Agent, Lawyer, Surveyor, Interior Designer...)

Search button on the right using the Houzii primary color.

2. Quick Category Filters (Below Search Bar)

Display professional service categories as quick selection chips or buttons.

Categories include:

• Agents
• Lawyers
• Surveyors
• Interior Designers
• Movers
• Property Inspectors
• Maintenance Services
• Cleaning Services
• Property Managers

These category chips act as quick filters and update the search results.

3. Filter Bar (Reuse Explore Page Filters)

Below the category filters, create a filter row similar to the property listing page.

Filters should include:

• Location
• Service category
• Rating
• Verified professionals
• Years of experience
• Sort by (Top Rated, Most Reviewed, Newest)

Include a Sort dropdown similar to the property explore page.

4. Professional Listing Grid

Display professionals using card-based layout similar to property cards but adapted for professionals.

Each professional card should include:

• Profile image or company logo
• Professional name
• Professional title or category (Agent, Lawyer, Surveyor, etc.)
• Rating and number of reviews
• Location
• Key services offered
• Verified badge if applicable

Example card layout:

Profile image
Name
Profession

⭐ 4.8 (125 reviews)

Location: Lagos

Services: Property law, contract review, title verification

Buttons:

• View Profile
• Contact Professional

5. Profile Interaction Behavior

When users click View Profile, open the professional profile page where they can view:

• Professional details
• Experience
• Services offered
• Portfolio (optional)
• Reviews and ratings

Viewing profiles should not require login.

6. Contact Button Behavior (Signup Funnel)

When a user clicks Contact Professional, check whether the user is logged in.

If the user is not logged in, trigger the existing Houzii authentication modal.

The modal should display:

Login or Create Account

Text example:

Create an account or log in to contact professionals and send service requests.

Buttons:

• Sign Up
• Log In

After authentication, allow the user to proceed with contacting the professional.

This interaction acts as a conversion funnel to encourage user signup before initiating communication.

7. Featured Professionals Section

Above the professional grid or within it, include a section titled:

Featured Verified Professionals

Highlight professionals who are:

• Verified
• Highly rated
• Active on the platform

These cards should have a featured badge.

8. Request a Service Section

Below the marketplace grid, include a section for users who are unsure which professional to contact.

Title:

Need Help Finding the Right Professional?

Text:

Submit a request and verified professionals can respond to your service needs.

CTA button:

Request a Service

This allows Houzii to collect leads for professionals.

9. Empty State

If no professionals match the search criteria, show a friendly message:

No professionals found for your search.

Suggestions:

• Try adjusting your filters
• Explore other service categories

11. Design Consistency Requirements

Ensure the page:

• Uses the same card styles as the property explore page
• Uses Houzii brand colors and typography
• Maintains spacing and UI consistency with the rest of the platform
• Supports responsive layout for desktop and mobile

12. Goal of the Page

The primary goal of the page is to allow users to:

• Discover trusted professionals
• Explore services available within the Houzii ecosystem
• Encourage user account creation before contacting professionals

This page acts as a service marketplace integrated within the Houzii property ecosystem
