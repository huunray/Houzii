— Simplified Houzii Hero Search (Nigeria Context)

Design an improved hero section for the Houzii real estate platform homepage optimized for Nigerian users.

The hero should prioritize fast and simple property discovery, focusing on how people typically search for property in Nigeria.

The search experience should be simple, location-first, and mobile friendly.

Brand Guidelines

Use the Houzii design system.

Primary color: #7B2D42
Accent color: #C94B60
Primary dark: #5E1F32

Font: Inter

Buttons should be pill-shaped.

Cards should have 16px border radius and soft shadows.

The design should feel:

clean
modern
premium
trustworthy

Navigation

Top navigation should include:

Houzii logo (left)

Main navigation:

Explore
Invest
Services
For Property Owners
For Agents

Right side:

Login
Primary CTA: Get Started

Hero Background

Use a high-quality luxury Nigerian home image, similar to properties found in Lekki or Banana Island.

Apply a dark gradient overlay from left to right to ensure the headline and search remain readable.

Hero Headline

Headline:

Find the Perfect Property — Buy, Rent or Stay Anywhere

Highlight Buy, Rent or Stay using the Houzii accent color.

Subtext:

Search verified homes, apartments and shortlets across Nigeria.

Main Search Component

The hero must contain a floating search card.

Card design:

white background
soft shadow
rounded corners (16px)

The search should prioritize simplicity.

Search Intent Tabs

Tabs should include:

Buy
Rent
Shortlet

These tabs change the search fields dynamically.

Buy and Rent Search Fields

When the Buy or Rent tab is selected, show the following fields:

Location
Bedrooms
Search button

Example layout:

[ Where in Nigeria? ]   [ Bedrooms ]   [ Search ]

Location input should support suggestions such as:

Lagos
Abuja
Port Harcourt
Lekki
Yaba
Ikoyi
Victoria Island

Bedrooms dropdown options:

Studio / Self Contain
1 Bedroom
2 Bedrooms
3 Bedrooms
4 Bedrooms
5+ Bedrooms

The Search button should use the Houzii primary color.

Shortlet Search Fields

When the Shortlet tab is selected, replace the filters with:

Location
Check-in date
Check-out date
Search button

Example layout:

[ Location ]   [ Check-in ]   [ Check-out ]   [ Search ]

Guest selection should appear later on the results page, not in the hero.

AI Search (Secondary Discovery)

Below the structured search include an AI search assistant.

Label:

Ask Houzii AI

Example placeholder:

“3 bedroom duplex in Lekki with a pool under ₦120M”

This input allows users to search using natural language.

AI Search Suggestions

Below the AI input show clickable suggestions such as:

3 bedroom duplex in Lekki under ₦120M
Shortlet apartments in Victoria Island
Luxury homes in Ikoyi
2 bedroom flats for rent in Abuja

Clicking a suggestion should trigger a search.

Popular Areas

Add clickable location chips below the search.

Example:

Popular Areas

Lekki Phase 1
Victoria Island
Ikoyi
Abuja CBD
Gbagada

These chips should trigger instant search.

Interaction Behavior

The search component should behave dynamically.

When switching tabs:

Buy → show location + bedrooms
Rent → show location + bedrooms
Shortlet → show location + dates

The transition should animate smoothly.

Explore Page Behavior

Advanced filters should not appear in the hero.

After users click Search, they should go to the Explore Properties page, where they can refine results using filters such as:

Price
Property Type
Bathrooms
Amenities
Parking
Furnishing
New Developments

Design Goal

The hero section should make it immediately clear that users can:

Buy homes
Rent apartments
Book shortlets

The search should feel:

fast
simple
intuitive

Users should be able to start a search in two clicks or less.
