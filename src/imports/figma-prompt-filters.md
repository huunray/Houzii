Dynamic Search Filters

The hero search must support dynamic filters based on the selected tab.

Tabs include:

Buy
Rent
Shortlet
Invest

Each tab should update the search fields and smart filters.

Buy shows:

Location
Price Range
Property Type
Bedrooms

Rent shows:

Location
Monthly Rent
Bedrooms
Furnishing

Shortlet shows:

Location
Check-in Date
Guests
Price Per Night

Invest shows:

Location
Investment Type
Minimum Investment
Expected ROI

Each tab should also update the smart filter chips and AI search suggestions.

The transition between tabs should be animated and smooth.

UX Principle Behind This

This design works because it matches the mental model of the user.

Buy → ownership filters
Rent → monthly cost filters
Shortlet → stay filters
Invest → financial filters

So the search becomes contextual instead of generic.
