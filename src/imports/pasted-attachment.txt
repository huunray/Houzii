3. Dynamic Filter Behavior

The filter system must change based on the selected tab.

When user clicks BUY

Visible filters become:

Price
Property Type
Bedrooms
Land Size
More Filters

Listings update to Properties for Sale.

When user clicks RENT

Visible filters change to:

Price (per year)
Property Type
Bedrooms
Furnished
More Filters

Listings update to Rental properties.

When user clicks SHORTLET

Visible filters change to:

Check-in
Check-out
Guests
Price per night
More Filters

Listings update to Shortlet listings.

5. Scroll Interaction

When users scroll:

The search + filters become sticky.

Sticky bar example:

[Location] [Filters] [Sort] [Map]

This prevents users from scrolling back to refine search.
