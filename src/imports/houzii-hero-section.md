Improve Houzii Hero Section

Improve the hero section of the Houzii real estate website homepage to make the search experience clearer, more intuitive, and visually modern.

The hero section should prioritize property discovery, with the search component being the most prominent element.

Use the following brand tokens:

Primary color: #7B2D42
Accent color: #C94B60
Primary dark: #5E1F32
Font: Inter
Button style: pill shaped
Card radius: 16px

Design style should feel modern, premium, and trustworthy, similar to high-end property platforms.

Navigation Bar

Include the Houzii navigation bar at the top:

Left side:

Houzii logo

Center navigation:

Home
Explore Properties
Invest
Services
For Property Owners
For Agents

Right side:

About
Help Center
Login
Primary CTA button: Get Started

Hero Background

Use a high quality luxury property image, preferably a modern Nigerian luxury home similar to houses found in Lekki or Banana Island.

Apply a dark gradient overlay from left to right so text remains readable.

Example gradient:

linear-gradient(
90deg,
rgba(0,0,0,0.65) 0%,
rgba(0,0,0,0.45) 35%,
rgba(0,0,0,0.10) 70%,
rgba(0,0,0,0) 100%
)

The left side should contain the headline and search interface.

Hero Headline

Headline:

Find the Perfect Property — Buy, Rent or Stay Anywhere

Highlight Buy, Rent or Stay using the Houzii accent color.

Subtext:

Search thousands of verified homes, apartments and shortlets across Nigeria.

Main Search Interface

Create a floating search card centered under the headline.

Card design:

white background

subtle shadow

16px rounded corners

soft elevation

Search Tabs

At the top of the card include tabs:

Buy
Rent
Shortlet

Active tab should use the Houzii primary color.

Primary Search Fields

Keep the main search simple and fast.

Fields should include:

Location (with location icon)

Price Range

Search Button

Search button should use the primary brand color.

More Filters Option

Include a secondary link:

More Filters →

Clicking this expands advanced filters such as:

Property Type
Bedrooms
Bathrooms
Amenities
Furnishing
New Development

These filters should expand below the main search fields.

Map Search Functionality

Add a secondary discovery option below the search inputs.

Label:

Explore Properties on Map

Include a small map preview card showing a city map with property markers.

Example cities:

Lagos
Abuja
Port Harcourt

CTA:

Open Map Search

This takes users to the map-based property exploration page.

The map search is important because many property users prefer location-based discovery.

AI Search Feature

Below the structured search include an AI search assistant.

Label:

Houzii AI Search

Input placeholder example:

Try searching:
“3 bedroom apartment in Lekki under ₦80M with a pool”

Include AI suggestion examples:

Apartments in Ikoyi under ₦200M
Shortlets in Lekki near the beach
2 bedroom houses for rent in Abuja

Style the AI search as a secondary input so it does not compete with the main search.

Visual Hierarchy

The layout should follow this order:

Headline
Subtext
Search tabs
Search inputs
Search button
AI search suggestion
Map search option

Ensure the search card is the dominant element of the hero section.

Micro Interactions

Include subtle UI interactions:

Tabs animate when switching
Search button hover effect
Map preview hover highlight
AI suggestions clickable

Overall Design Goal

The hero should communicate:

fast property discovery
trust and professionalism
modern real estate platform
powerful search capabilities

The user should immediately understand they can:

search properties
explore on map
use AI to discover listings.
